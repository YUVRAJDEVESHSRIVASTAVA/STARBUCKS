# Starbucks Landing Page Clone

A responsive Starbucks-inspired landing page built with pure HTML, CSS, and JavaScript.

This project showcases a modern single-page design with a hero section, product cards, delivery/app sections, and a mobile-friendly navigation menu.

## Preview

- Brand-style landing page UI
- Smooth section-based scrolling
- Responsive layout for desktop, tablet, and mobile
- Mobile hamburger menu interaction

## Tech Stack

- `HTML5`
- `CSS3`
- `JavaScript (Vanilla)`
- `Boxicons` (for iconography)

## Features

- Fixed top navigation bar
- Animated hero image
- Shop cards with ratings and pricing
- Delivery and app promotion sections
- About and social contact footer
- Responsive breakpoints for multiple screen sizes

## How To Run Locally

1. Clone or download this repository.
2. Open the folder in VS Code.
3. Open `Starbucks/index.html` in your browser.

Quick option in VS Code:
- Install the **Live Server** extension.
- Right-click `Starbucks/index.html` and select **Open with Live Server**.
